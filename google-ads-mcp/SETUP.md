# Google Ads MCP Server Setup

## Prerequisites

- Python 3.12+
- Google Cloud project with Google Ads API enabled
- Google Ads account with developer token
- OAuth 2.0 credentials (Desktop application)

## Quick Start

### 1. Install Dependencies

```bash
cd google-ads-mcp
pip install -e .
```

### 2. Set Up Authentication

```bash
python -m hogans_ads_mcp.auth
```

This will:
- Open a browser for Google OAuth sign-in
- Save your refresh token to `google-ads.yaml`

### 3. Configure Credentials

If you prefer manual setup, copy the example config:

```bash
cp google-ads.yaml.example google-ads.yaml
```

Fill in your credentials:
- **client_id** / **client_secret**: From Google Cloud Console
- **refresh_token**: Generated by the auth script
- **developer_token**: From Google Ads UI > Tools > API Center
- **login_customer_id**: Your manager account ID (no dashes)

### 4. Run the MCP Server

```bash
python -m hogans_ads_mcp.server
```

### 5. Connect with Claude Code

Copy `.mcp.json` to your project root, or add to your Claude Code settings:

```json
{
  "mcpServers": {
    "hogans-google-ads": {
      "command": "python",
      "args": ["-m", "hogans_ads_mcp.server"],
      "cwd": "google-ads-mcp",
      "env": {
        "GOOGLE_ADS_CREDENTIALS": "google-ads-mcp/google-ads.yaml"
      }
    }
  }
}
```

## Available Tools

### Campaign Management
- **list_campaigns** - List all campaigns with status and budget
- **get_campaign** - Get detailed campaign info with metrics
- **create_campaign** - Create a new campaign (Search, Display, or Discovery)
- **update_campaign_status** - Enable, pause, or remove a campaign
- **update_campaign_budget** - Change a campaign's daily budget

### Reporting
- **get_campaign_performance** - Performance metrics across campaigns
- **get_ad_group_performance** - Ad group level breakdown
- **get_search_terms_report** - See actual search queries triggering ads
- **get_geo_performance** - Geographic performance breakdown

### Keyword Management
- **list_keywords** - List keywords in an ad group
- **add_keywords** - Add keywords with match type
- **add_negative_keywords** - Add negative keywords to prevent wasted spend
- **get_keyword_performance** - Keyword-level metrics with quality score

### Ad Management
- **create_ad_group** - Create ad groups within campaigns
- **create_responsive_search_ad** - Create responsive search ads
- **list_ads** - List all ads in a campaign
- **update_ad_status** - Enable, pause, or remove ads
